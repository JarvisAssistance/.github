# 쟈비스 비서 프로젝트

- 실제 실행시킬 환경
	- mac mini m4, ram 32gb
- 사용할 추론모델
	- 로컬에서 실행하는 ollama gemma3

- 제작할 기능
1. wakeword 설정
2. voice cloning 으로 TTS 지원
3. 자연스러운 대화 지원 (말을 중간에 끊는것도 가능해야함)

- 추후 제작할 기능
1. 날씨 정보 제공
2. 검색 (검색 결과중에 선택할 수 있게 할 예정)
3. 메모

- 사용할 개발 스택
0. python version : 3.9
1. 파이썬 가상환경 venv 사용 (디렉토리 이름 venv)
